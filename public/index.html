      vMix overlays

      Transparent 1920×1080 graphics for vMix Web Browser inputs. Data comes from RowIT CSV feeds and AHD_lookup (club logos, event names). Set the regatta code above — overlays read the same code via ?regatta= or this page’s saved value.

      Live stream graphics guide

      Live graphic race

      Sets the race for draw, lower third, and results on open vMix overlays. Leader lane applies when the leader graphic (w) is on air. Use + / − to step through the daysheet.

      In vMix, focus the Milford browser input: d/l/r/t play in (3s then text), g speed dot + TimeTracker video, o out (reverse), c clear. Configure speed device on the fleet map first.

      Buttons send triggers to open Milford overlay tabs on this site. Use vmix-rnz-milford.html for regatta graphics and speed overlay.
